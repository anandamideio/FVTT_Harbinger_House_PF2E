# Harbinger House PF2E Module
[![CI](https://github.com/anandamideio/FVTT_Harbinger_House_PF2E/actions/workflows/ci.yml/badge.svg)](https://github.com/anandamideio/FVTT_Harbinger_House_PF2E/actions/workflows/ci.yml)

## Why did I make this?
A friend recently mentioned how they miss some of our adventure's in Sigil, and, so I though to myself, "Well, I own this adventure, so why not try and convert it to Pathfinder 2e? Seems like a solid chance to see the work that goes into a adventure module".

### What was the process like?
I started by setting up an OCR pipeline to extract the text from my PDF, and then spent some time cleaning it up, and converting the extracted text to markdown. Then I wrote a script to convert all of the items, NPCs, spells, and hazards into JSON format that I could use in this module. Then came the fun and hard part, converting from dnd 2e to pf2e (this was a fun challenge, seeing what had a one to one, and what I need to create custom for this adventure.).
